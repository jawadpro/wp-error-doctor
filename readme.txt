An SEO-ready WordPress security, speed, error, and website health checker with qualified lead capture.

== Description ==

WP Error Doctor adds a branded floating diagnostic balloon to your website. Visitors can scan a public website, review basic evidence, and send the report to Jawad for professional WordPress support.

The scanner uses WordPress safe HTTP requests, never requests credentials, does not log in, and treats protected or renamed admin paths as inconclusive.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/ or install its ZIP from Plugins > Add New.
2. Activate WP Error Doctor Lead Widget.
3. Open Settings > WP Error Doctor.
4. Configure the lead email, accent color, widget position, and text.
5. Confirm that WordPress email delivery is configured with SMTP.
